Linux · C++ · HTTP/1.1

High Performance
C++ Web Server

VServer 是一个从零实现的高性能 C++ Web Server， 基于 epoll、线程池、HTTP 状态机、时间轮和异步日志系统构建， 用于学习 Linux 网络编程与后端服务器核心机制。

IO

Reactor Core

主线程基于 epoll 负责事件分发，使用非阻塞 I/O 处理高并发连接。

FSM

HTTP Parser

使用状态机解析请求行、请求头和请求体，支持半包、粘包、Pipeline 与 Keep-Alive。

LOG

Async Logger

自定义异步日志系统，支持日志等级、时间戳、后台写入和日志滚动。
